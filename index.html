Simon Say's Game

Press "Play" Button To Start The Game :)


    Gameplay Tips For Playing Game

  • The computer program displays a sequence of colored lights.
  • The player must repeat the sequence by pressing the corresponding colored buttons in the correct order.
  • If the player successfully repeats the sequence, the computer program will add another color to the sequence.
  • The game continues until the player makes a mistake.
  • The player's score is based on the number of rounds they completed.
